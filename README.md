# MLP
MLP NN for yahoo finance for take-profit/stop-loss strategy
Files:
1. work_with_yfinance.py - using for load data from yahoo finance
2. load_data.py - using for converting the data from yahoo finance to correct format for pytorch Dataset
3. Dataset.py - using for create pytorch Dataset with the data from yahoo finance
4. MLP.py - main file with NN
5. test_NN.py - testing our NN on some piece of our dataset
